— Rules —
サーバー利用規約
01
🤝 挨拶とマナー

相手を尊重する：攻撃的な口調や煽りは禁止です。

不快な発言の禁止：差別・過度な下ネタ・特定個人への中傷は控えてください。

02
📂 チャンネルの使い分け

各チャンネルの趣旨に沿った投稿をお願いします。

意味のない連投・スタンプの大量送信は禁止です。

03
🔒 プライバシーと自衛

本名・住所・電話番号など個人特定に繋がる情報の投稿は禁止です。

出会い目的のDM送信・勧誘は厳禁です。

04
🔔 メンションの扱い

急用でもないのに管理者や特定メンバーへメンションを飛ばしすぎるのは避けましょう。

05
🚫 禁止事項

荒らし行為：サーバー運営を妨害するすべての行為。

宣伝：他サーバーやSNSの宣伝は専用チャンネル以外では禁止です。

📌 管理者からのお願い

ルールに違反している人を見かけたら、直接反論せずに「お問い合わせ」または「管理人へのDM」で報告してください。平和な環境作りにご協力をお願いします！

— おいしいの歴史 —
私たちの軌跡
2023年
🎮 IKSクランの時代

ゆずぴー・じつねこ・いくすわ・すーは、IKSというクランで出会い、2023年頃から仲を深めた。ゲームを通じて絆を築いていったが、やがてIKSは崩壊。それぞれは一時、疎遠になっていく。

2025年8月
🌸 再会とサーバー誕生

じつねこがTikTokを見ていたある日、ゆずぴーを偶然発見。コメントでやり取りが始まり「俺たちでサーバー作ろう」という話になった。

初代サーバー
poop.exe

おいしいコミュニティの原点。最初、じつねこはメンバーとして参加。管理者はすー・ゆずぴー・いくすわ・ゆきみの4名だった。

poop.exe 時代
⚡ 管理者の交代と成長

ゆきみによる職権乱用の告発をきっかけに管理者体制が変わり、じつねこがサーバーを引っ張る管理者として昇格。メンバーも増え、サーバーは発展していった。

poop.exe 末期
🚨 鯖制限・荒らし・移行

突如として鯖制限が発動。画像も送れず、誰も入れない状態になりサーバーの移行を余儀なくされた。その混乱の中、えだまめという人物による荒らし被害も発生し、以降は疎遠となった。

移行後
✨ 美味しいコミュニティの誕生
第2代サーバー
美味しいコミュニティ

メンバー公募はせず、管理者は固定制。新たな体制で再スタートを切った。

移行直後
💔 「姫」事件とすーのNuke

新サーバーになってまもなく、鯖民「姫」が自傷行為をほのめかし、コミュニティ全体を巻き込む事態となった。それに呆れたすーは美味しい鯖をNukeし、縁を切った。

1000人規模時代
💥 じつねこ管理者剥奪事件

メンバーが1000人規模に成長したころ、事件が起きた——じつねこの管理者剥奪。激怒したじつねこはサーバーを退出し、全員と疎遠になった。「みんなで作ったのに、どうして」と嘆くメンバーも多かった。しかし最終的には許され、じつねこは管理者として復帰した。

2000人時代
👑 ゆきみの管理者復帰と新たな課題

ゆきみが管理者として復帰した。復帰自体はめでたいことだったが、アナウンスでのエブリワン（@everyone）の多用が問題に。浮上勢はだんだん限られてきてしまった。

3000人突破前夜
🎮 ゆずぴーの大バズり・60万再生
大事件
📹 60万再生の衝撃

ゆずぴーはRobloxのpoophubの制作者。その頭脳を活かし、有名アニメのワンシーンの技をRobloxのチートで再現した動画が爆発的にバズり、60万再生を記録。これが3000人突破の大きなカギとなった。

2026年
🎉 3000人突破・そして移行へ

ついにメンバーが3000人を突破。しかし、所有権を持ついくすわのアカウントに登録されていたメールアドレスが死亡していたことが判明。3000人突破を機に新サーバーへの移行を決断。旧サーバーは無法地帯と化した。

現在
🍽️ おいしいコミュニティ、今へ

数々の事件と試練を乗り越えた私たちは、雑談とショップを両軸にした現在の「おいしいコミュニティ」へと辿り着いた。これからも、おいしい時間を一緒に作っていこう。
